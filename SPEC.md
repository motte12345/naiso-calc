# SPEC.md — 内装材料カリキュレーター

## サイト概要

- **目的**: DIY・リフォームで必要な内装材料の量を計算するツール群
- **収益モデル**: AdSense + Amazon/楽天アフィリエイト（計算結果横に関連商品表示）
- **技術構成**: React（静的サイト）、Vite + TypeScript、Vercel or Cloudflare Pages

## 差別化ポイント

- 窓・ドアを除外できる（既存サイトは部屋全体面積のみが多い）
- 柄リピート・目地幅など実用パラメータ対応
- 副資材（接着剤・下地材等）も表示し買い忘れ防止
- スマホで現場から使えるレスポンシブ設計

## 計算ツール仕様

### 1. 壁紙（クロス）必要量計算

**入力**:
- 部屋の幅(m)・奥行(m)・高さ(m)
- 窓の数とサイズ（幅×高さ）
- ドアの数とサイズ（幅×高さ）

**出力**:
- 必要なm数
- ロール数（国産920mm幅 / 輸入530mm幅を選択可）
- 柄リピートによるロス分の加算

**補足表示**:
- のり・パテ・ローラー等の副資材の目安量

**計算ロジック**:
1. 壁面積 = (幅×2 + 奥行×2) × 高さ
2. 開口部面積 = Σ(窓の幅×高さ) + Σ(ドアの幅×高さ)
3. 有効面積 = 壁面積 - 開口部面積
4. ロール幅で割って必要巾数を算出
5. 1巾あたりの長さ = 高さ（+ 柄リピートがある場合はリピート単位で切り上げ）
6. 必要m数 = 巾数 × 1巾あたりの長さ
7. ロール数 = 必要m数 ÷ 1ロールの長さ（国産50m / 輸入10m）で切り上げ

**副資材目安**:
- のり: 有効面積 × 150g/m² → kg
- パテ: 有効面積 × 0.3kg/m² → kg
- ジョイントローラー: 1個

### 2. 床材（フローリング・CF）必要量計算

**入力**:
- 部屋の幅(m)・奥行(m)
- 床材の規格（1枚の幅×長さ, mm単位）
- 貼り方（並行貼り / 斜め貼り）

**出力**:
- 必要枚数 or m²数
- ロス率加算済み（並行5% / 斜め10%）

**補足表示**:
- 下地材・接着剤の目安

**計算ロジック**:
1. 床面積 = 幅 × 奥行
2. 1枚面積 = 板幅 × 板長さ（mm→m²変換）
3. 理論枚数 = 床面積 ÷ 1枚面積（切り上げ）
4. ロス率加算: 並行→×1.05 / 斜め→×1.10
5. 推奨枚数 = 理論枚数 × ロス率（切り上げ）

**副資材目安**:
- フローリング接着剤: 床面積 × 0.5kg/m² → kg
- CF用接着剤: 床面積 × 0.3kg/m² → kg

### 3. タイル必要枚数計算

**入力**:
- 施工面の幅(mm)・高さ(mm)
- タイル1枚のサイズ（幅×高さ, mm）
- 目地幅(mm)

**出力**:
- 必要枚数
- 予備分（+10%）込みの枚数
- 目地材の必要量(kg)

**計算ロジック**:
1. 横枚数 = 施工幅 ÷ (タイル幅 + 目地幅)（切り上げ）
2. 縦枚数 = 施工高さ ÷ (タイル高さ + 目地幅)（切り上げ）
3. 必要枚数 = 横 × 縦
4. 予備込み = 必要枚数 × 1.10（切り上げ）
5. 目地材 = 目地の体積(mm³→L) × 目地材密度(約1.5kg/L)

**目地材体積の計算**:
- 横目地の長さ合計 × 目地幅 × タイル厚み(デフォルト7mm)
- 縦目地の長さ合計 × 目地幅 × タイル厚み
- 合計体積 → kg換算

### 4. ペンキ・塗料の必要量計算

**入力**:
- 塗装面積（直接入力 or 部屋寸法から算出）
- 部屋寸法入力時: 幅・奥行・高さ、窓・ドア除外
- 塗り回数（1〜3回）

**出力**:
- 必要リットル数
- 缶サイズ別の必要個数（0.7L / 1.6L / 4L）

**計算ロジック**:
1. 塗装面積を算出（直接入力 or 壁紙と同様のロジック）
2. 1回あたり必要量 = 面積 ÷ 塗布面積(8m²/L) ※一般的な水性塗料
3. 合計必要量 = 1回分 × 塗り回数
4. 缶サイズ別: 合計÷缶サイズ（切り上げ）を各サイズで表示

**副資材目安**:
- マスキングテープ: 部屋周長 × 塗り回数分
- ローラー: 1セット
- 養生シート: 床面積分
